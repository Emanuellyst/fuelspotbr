Projeto Acadêmico

Sistemas de Informação

O FuelSpotBR é um sistema modelo desenvolvido como trabalho acadêmico do curso de Sistemas de Informação, com o objetivo de apresentar uma solução para comparação de preços de combustíveis em postos de gasolina.

Os dados exibidos são obtidos por meio de coletas realizadas pelos próprios desenvolvedores, permitindo simular um cenário real de análise e organização de informações.

Este projeto aplica conceitos de desenvolvimento web, design de interfaces e experiência do usuário, demonstrando na prática como sistemas de informação podem ser utilizados para resolver problemas do cotidiano.

Objetivo

Desenvolver uma plataforma que permita aos usuários pesquisar e comparar preços de combustíveis, facilitando a escolha do posto mais vantajoso.

Público-Alvo

Motoristas e consumidores que desejam economizar no abastecimento por meio da comparação de preços entre diferentes postos.

Diferencial

Sistema de busca com filtros por nome, cidade, bairro e tipo de combustível, além do destaque automático do menor preço entre os resultados.

Funcionalidades Principais

Busca de Postos

Permite pesquisar postos pelo nome para localizar com mais facilidade o estabelecimento desejado.

Filtros de Pesquisa

O sistema permite refinar a busca por cidade, bairro e tipo de combustível, facilitando a localização dos resultados.

Comparação de Preços

Os postos encontrados são exibidos com os valores dos combustíveis para ajudar na comparação entre as opções disponíveis.

Destaque do Melhor Preço

Entre os resultados exibidos, o sistema identifica e destaca automaticamente o posto com o menor preço do combustível selecionado.
